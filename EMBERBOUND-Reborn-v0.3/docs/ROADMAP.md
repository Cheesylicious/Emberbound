# Roadmap

## 0.1 – Spielbarer Prolog

- [x] neue 32‑Bit-Pixelrichtung
- [x] Dorf, Weltkarte, Wald und Reliquiar
- [x] fließende Bewegung und Kollision
- [x] Storydialoge und drei Gruppenmitglieder
- [x] animiertes Rundensystem
- [x] erste Drachenform
- [x] Speichern, Laden und Export
- [x] Tastatur und Touch
- [x] native Mac-Verpackung

## 0.2 – Szenischer Kartenumbau

- [x] sichtbare Platzhalterkacheln vollständig entfernt
- [x] drei detaillierte pseudo-isometrische Erkundungskarten
- [x] organische Laufzonen statt sichtbarem Blockraster
- [x] Brücken, Plattformen, Abgründe, Häuser und Brunnen korrekt kollidierend
- [x] automatische Migration alter Speicherpositionen
- [x] vollständige Wegtests zu Storyziel, Ausgängen und Boss

## 0.3 – Charakter- und Progressionssysteme

- [x] neue konsistente Helden-Sprite-Sheets mit Blick- und Laufposen
- [x] Dialogporträts für Ryn, Liora, Bram, Mara und die Brut
- [x] individuelle Wachstumskurven und echte Stufenanstiege
- [x] Inventar mit LP-, SP- und DP-Gegenständen
- [x] Waffen-, Rüstungs- und Zubehörplätze pro Figur
- [x] Ausrüstungsboni verändern Kampfwerte und Maximalressourcen
- [x] Gegnerbeute und Mehrgegnerkämpfe
- [x] Gegenstände als reguläres Kampfkommando

## 0.2 – Vollständiges Kapitel I

- Ausrüstung, Inventarbenutzung und Händler
- Statuswerte, Elemente und Schwächenanzeige
- acht statt vier Gegnertypen
- zwei zusätzliche Mini-Bosse
- Innenräume in Kohlgrund
- Musik- und Geräuschpaket
- mehr Richtungs- und Laufphasen pro Figur

## 0.3 – Systeme der Vollversion

- Brut-Gen-Kombinationen
- lernbare Gegnertechniken
- Meistersystem
- Formation und Gruppenwechsel
- Angel- und Lagerplatzsystem
- Questjournal und optionale Erinnerungsorte
- mehrere Speicherplätze

## 0.5 – Mobile Alpha

- Capacitor-Projekte für iOS und Android
- native sichere Speicher-Bridge
- frei positionierbare Touch-Steuerung
- Geräteprofile für 16:9, 19.5:9 und Tablets
- Unterbrechungs-, Akku- und Performance-Tests

## 1.0 – Vollversion

- fünf Kapitel und Finale
- mindestens 20 Stunden Hauptgeschichte
- sechs spielbare Figuren
- zwölf kombinierbare Brut-Gene
- drei Endzustände
- Barrierefreiheit, Lokalisierung und Store-QA

## Release-Grundsätze

Jede Version muss alte Spielstände migrieren oder verständlich als inkompatibel
kennzeichnen. Neue Grafik wird auf Bodenhaftung, klare Silhouette und
Pixelkonsistenz geprüft. Vor einem kommerziellen Release werden alle Assets,
Schriften, Klänge und Werkzeuge erneut lizenzrechtlich auditiert.
